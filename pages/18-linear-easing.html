Aba 18: linear() easing function

Nova função de easing linear() que permite criar curvas complexas com múltiplos stops. Alternativa ao cubic-bezier() para alguns casos. Chrome 113+

linear() vs linear keyword

/* linear keyword (constante) */
animation-timing-function: linear;

/* linear() com 1 stop intermediário */
animation-timing-function: linear(0, 0.5 25%, 1);
/* equivalente a ease */

/* linear() com múltiplos stops */
animation-timing-function: linear(
  0,
  0.25 20%,
  0.5 50%,
  0.75 80%,
  1
);

Bounce com linear()

/* Recriar bounce sem cubic-bezier */
@keyframes bounce {
  0%   {
    bottom: 2px;
    animation-timing-function: linear(
      0,
      0.0625 12.5%,
      0.25 25%,
      0.5625 37.5%,
      1 50%,
      0.5625 62.5%,
      0.25 75%,
      0.0625 87.5%,
      0
    );
  }
  50%  { bottom: calc(100% - 42px); }
  100% { bottom: 2px; }
}

Pulse Pattern

/* Pulsa entre 0 e 1 várias vezes */
animation-timing-function: linear(
  0,
  1 25%,
  0 50%,
  1 75%,
  0
);

Material Standard

/* Aproximação Material Design
   standard easing via linear() */
animation-timing-function: linear(
  0,
  0.05 20%,
  0.1 30%,
  0.2 40%,
  0.3 50%,
  0.4 60%,
  0.5 70%,
  0.6 80%,
  0.85 95%,
  1
);

Step-like (não-strict)

/* Step-like motion com
   2 pontos fixos + transições */
animation-timing-function: linear(
  0,
  0.5,
  0.5,
  1
);
/* vai de 0 → 0.5 instantaneamente,
   depois linearmente até 1,
   e fica em 0.5 no meio */

Browser Support

Função moderna. Use com @supports como fallback.

@supports (animation-timing-function: linear(0, 1)) {
  .box {
    animation-timing-function:
      linear(0, 0.5 25%, 1);
  }
}
