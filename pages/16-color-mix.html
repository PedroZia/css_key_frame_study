Aba 16: color-mix() animado

color-mix(in <color-space>, cor1 <%>, cor2) interpola entre duas cores em diferentes espaços de cor. Combinado com @property, permite transições suaves entre cores. Chrome 111+

Background animado

Hover: 0% → 100% do mix

@property --amount {
  syntax: '<percentage>';
  initial-value: 0%;
  inherits: false;
}
.box {
  --from: #ffffff;
  --to: #c0c0c0;
  --amount: 0%;
  background: color-mix(in srgb,
              var(--from) var(--amount),
              var(--to));
  transition: --amount 600ms ease;
}
.box:hover { --amount: 100%; }

Texto com cor interpolada

CSS Color Mix

Hover: ciano → rosa em oklab

.text {
  --c1: #ffffff;
  --c2: #404040;
  --p: 0%;
  color: color-mix(in oklab,
            var(--c1) var(--p),
            var(--c2));
  transition: --p 500ms ease;
}
.text:hover { --p: 100%; }

Border com mix em oklch

Border Animada
.box {
  --c1: #ffffff;
  --c2: #c0c0c0;
  --amt: 0%;
  border: 3px solid color-mix(
            in oklch,
            var(--c1) var(--amt),
            var(--c2));
  transition: --amt 500ms ease;
}
.box:hover { --amt: 100%; }

Box-shadow com cor dinâmica

Shadow Glow
.box {
  --shadow: #ffffff;
  --op: 0%;
  box-shadow: 0 0 30px color-mix(
                in srgb,
                var(--shadow) var(--op),
                transparent);
  transition: --op 400ms ease;
}
.box:hover { --op: 100%; }

Color Spaces Comparison

/* Diferentes espaços produzem
   interpolações visuais diferentes */
color-mix(in srgb,  #ffffff, #c0c0c0);
color-mix(in oklab, #ffffff, #c0c0c0);
color-mix(in oklch, #ffffff, #c0c0c0);
color-mix(in hsl,   #ffffff, #c0c0c0);
color-mix(in lch,   #ffffff, #c0c0c0);

/* oklab/oklch = perceptualmente uniforme
   (melhor para gradientes suaves) */

Mix aninhado (3 cores)

Hover: redistribui as proporções

@property --a { syntax: '<percentage>'; initial-value: 33%; inherits: false; }
@property --b { syntax: '<percentage>'; initial-value: 33%; inherits: false; }
@property --c { syntax: '<percentage>'; initial-value: 34%; inherits: false; }

.box {
  background: color-mix(in srgb,
    color-mix(in oklab,
      #ffffff var(--a),
      #c0c0c0) var(--b),
    #808080 var(--c));
  transition: --a 500ms, --b 500ms, --c 500ms;
}
.box:hover { --a: 10%; --b: 80%; --c: 10%; }
