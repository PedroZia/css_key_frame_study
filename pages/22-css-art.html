Aba 22: CSS Art Gallery

Animações decorativas puras para inspiração. Sem propósito funcional — apenas para ver as possibilidades do CSS.

Pulse Rings

.pulse {
  width: 60px; height: 60px;
  background: var(--accent);
  border-radius: 50%;
  position: relative;
}
.pulse::before, .pulse::after {
  content: '';
  position: absolute;
  inset: 0;
  border-radius: 50%;
  border: 2px solid var(--accent);
  animation: pulse 2s ease-out infinite;
}
.pulse::after { animation-delay: 1s; }
@keyframes pulse {
  0%   { transform: scale(1);   opacity: 1; }
  100% { transform: scale(2.5); opacity: 0; }
}

Floating Shapes

@keyframes float {
  0%, 100% { transform: translate(0,0)    rotate(0); }
  25%      { transform: translate(20px,-30px) rotate(90deg); }
  50%      { transform: translate(-15px,-50px) rotate(180deg); }
  75%      { transform: translate(-30px,-20px) rotate(270deg); }
}

Wave Bars

.bar { animation: wave 1.5s infinite; }
.bar:nth-child(n) {
  animation-delay: calc(n * 0.1s);
}
@keyframes wave {
  0%, 100% { height: 20%; }
  50%      { height: 90%; }
}

Morphing Blob

@keyframes blob {
  0%, 100% { border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%; }
  25%      { border-radius: 30% 60% 70% 40% / 50% 60% 30% 60%; }
  50%      { border-radius: 50% 60% 30% 60% / 30% 60% 70% 40%; }
  75%      { border-radius: 60% 40% 60% 30% / 60% 40% 30% 70%; }
}

Orbital System

.planet {
  animation: orbit 4s linear infinite;
  transform-origin: 0 0;
}
@keyframes orbit {
  from { transform: rotate(0) translateX(70px); }
  to   { transform: rotate(360deg) translateX(70px); }
}

DNA Helix

.dot { animation: dna 2s infinite; }
.dot:nth-child(odd)  { animation-name: dna1; }
.dot:nth-child(even) { animation-name: dna2; }
.dot:nth-child(n) { animation-delay: n * 0.1s; }
@keyframes dna1 { 50% { transform: translateX(30px); } }
@keyframes dna2 { 50% { transform: translateX(-30px); } }

Spinning Squares

.sq { animation: spin 4s linear infinite; }
.sq:nth-child(1) { animation-duration: 4s; }
.sq:nth-child(2) {
  animation-duration: 3s;
  animation-direction: reverse;
}
.sq:nth-child(3) { animation-duration: 5s; }
@keyframes spin {
  to { transform: rotate(360deg) scale(1.2); }
}

Aurora

.aurora::before {
  content: '';
  position: absolute;
  width: 200%; height: 200%;
  top: -50%; left: -50%;
  background: conic-gradient(from 0deg,
    transparent, var(--accent),
    var(--accent-2), transparent 30%);
  animation: spin 4s linear infinite;
  filter: blur(40px);
}
