Aba 14: offset-path animations

offset-path define um caminho (path SVG, circle, ray) ao longo do qual o elemento se move. offset-distance controla a posição (0%–100%). offset-rotate: auto rotaciona o elemento para seguir a direção do path.

Path SVG — Curva fechada

.ball {
  offset-path: path('M 20,110
    C 20,40 200,40 200,110
    C 200,180 20,180 20,110 Z');
  animation: move 4s ease-in-out infinite;
}
@keyframes move {
  0%   { offset-distance: 0%; }
  50%  { offset-distance: 50%; }
  100% { offset-distance: 100%; }
}

offset-rotate: auto

.arrow {
  offset-path: path('M 10,40 Q 150,10 290,40
                              T 290,160');
  offset-rotate: auto;
  /* elemento rotaciona seguindo
     a tangente do path */
  animation: follow 5s linear infinite;
}

Com rotação customizada

.star {
  offset-path: path('...');
  offset-rotate: auto;
  animation: spinMove 8s linear infinite;
}
@keyframes spinMove {
  from {
    offset-distance: 0%;
    offset-rotate: 0deg;
  }
  to {
    offset-distance: 100%;
    offset-rotate: 360deg;
  }
}

offset-path: ray()

.square {
  offset-path: ray(45deg closest-side);
  /* raio a 45° a partir do centro
     do elemento, até a borda
     mais próxima do container */
  animation: rayMove 3s ease-in-out
             infinite alternate;
}
@keyframes rayMove {
  from { offset-distance: 0%; }
  to   { offset-distance: 100%; }
}

offset-path: circle()

.planet {
  offset-path: circle(80px at 50% 50%);
  /* círculo de raio 80px
     centrado em 50% 50% */
  offset-rotate: auto;
  animation: orbit 5s linear infinite;
}
@keyframes orbit {
  from { offset-distance: 0%; }
  to   { offset-distance: 100%; }
}

Path S — Snake motion

.snake {
  offset-path: path('M 10,100
    C 50,10 150,10 150,100
    S 250,190 290,100');
  animation: slither 4s ease-in-out
             infinite;
}
@keyframes slither {
  0%, 100% { offset-distance: 0%; }
  50%      { offset-distance: 100%; }
}
