Aba 21: backdrop-filter & Glassmorphism

backdrop-filter aplica filtros (blur, saturate, etc) ao que está ATRÁS do elemento. Combinar com fundo translúcido cria o efeito "vidro fosco" (glassmorphism). Chrome 76+

Glassmorphism Básico

📱 Card com glass effect
.glass {
  background: rgba(255, 255, 255, .15);
  backdrop-filter: blur(20px)
                  saturate(180%);
  -webkit-backdrop-filter: blur(20px)
                          saturate(180%);
  border: 1px solid
          rgba(255, 255, 255, .2);
  border-radius: 12px;
}

Hover intensifica o blur

Hover me
.card {
  backdrop-filter: blur(8px)
                  saturate(150%);
  transition: backdrop-filter 400ms;
}
.card:hover {
  backdrop-filter: blur(20px)
                  saturate(200%);
}

backdrop-filter: hue-rotate

Hue Rotate
/* Múltiplos filtros encadeados */
backdrop-filter:
  blur(12px)
  hue-rotate(45deg)
  invert(15%);

/* Equivalente a filter mas
   aplicado ao backdrop */

Blur Animado

Animando blur
@keyframes animBlur {
  0%, 100% {
    backdrop-filter: blur(4px)
                    brightness(1);
  }
  50% {
    backdrop-filter: blur(16px)
                    brightness(1.3);
  }
}
.glass {
  animation: animBlur 3s ease
             infinite;
}

Combo: blur + saturate + contrast

Triple Filter
.glass {
  background: rgba(255, 255, 255, .15);
  backdrop-filter:
    blur(14px)
    saturate(180%)
    contrast(120%);
  -webkit-backdrop-filter:
    blur(14px)
    saturate(180%)
    contrast(120%);
  border: 1px solid
          rgba(255, 255, 255, .25);
}

Performance Tips

backdrop-filter é custoso. Use com moderação.

/* ✅ Bom: blur pequeno + área limitada */
.card {
  backdrop-filter: blur(8px);
}

/* ❌ Evite: blur enorme em área grande */
.fullscreen {
  backdrop-filter: blur(40px);
}

/* Use will-change para avisar o browser */
.glass {
  will-change: backdrop-filter;
}

/* Limite blur a ~20px em mobile */
