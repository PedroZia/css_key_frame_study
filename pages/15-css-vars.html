Aba 15: CSS Variables em @keyframes

Usar var() dentro de @keyframes e transition. Requer @property para que o browser saiba interpolar. Sem @property, a mudança é instantânea (não anima).

Rotação animada com hover

Passe o mouse no card

@property --rot {
  syntax: '<angle>';
  initial-value: 0deg;
  inherits: false;
}
.box {
  --rot: 0deg;
  transform: rotate(var(--rot));
  transition: --rot 800ms ease;
}
.card:hover .box {
  --rot: 360deg;
}

Escala com spring

1.3x

Hover: scale 1 → 1.3 com cubic-bezier spring

@property --s {
  syntax: '<number>';
  initial-value: 1;
  inherits: false;
}
.box {
  --s: 1;
  transform: scale(var(--s));
  transition: --s 400ms
              cubic-bezier(.17,.89,.32,1.28);
}
.box:hover { --s: 1.3; }

Bounce com @keyframes + var

@property --y {
  syntax: '<length>';
  initial-value: 0px;
  inherits: false;
}
.ball {
  --y: 0px;
  transform: translateY(var(--y));
  animation: bounce 2s infinite;
}
@keyframes bounce {
  0%, 100% { --y: 0px;   }
  50%      { --y: -30px; }
}

Múltiplas vars combinadas

CSS

Hover: rotaciona, move, muda matiz

@property --r   { syntax: '<angle>';  initial-value: 0deg; inherits: false; }
@property --x   { syntax: '<length>'; initial-value: 0px;  inherits: false; }
@property --hue { syntax: '<integer>'; initial-value: 0;   inherits: false; }

.box {
  --r: 0deg; --x: 0px; --hue: 0;
  transform: translateX(var(--x)) rotate(var(--r));
  background: hsl(var(--hue), 80%, 60%);
  transition: --r 500ms, --x 500ms, --hue 800ms;
}
.box:hover {
  --r: 45deg; --x: 30px; --hue: 180;
}

Gradient animado via vars

Hover: gradient muda de cor suavemente

@property --c1 { syntax: '<color>'; initial-value: #ffffff; inherits: false; }
@property --c2 { syntax: '<color>'; initial-value: #c0c0c0; inherits: false; }

.box {
  background: linear-gradient(135deg, var(--c1), var(--c2));
  transition: --c1 600ms, --c2 600ms;
}
.box:hover {
  --c1: #808080;
  --c2: #404040;
}

stroke-dashoffset animado

@property --len {
  syntax: '<number>';
  initial-value: 251;
  inherits: false;
}
circle {
  stroke-dasharray: var(--len);
  stroke-dashoffset: var(--len);
  animation: draw 2s ease infinite alternate;
}
@keyframes draw {
  from { --len: 251; }
  to   { --len: 0;   }
}
