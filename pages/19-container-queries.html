Aba 19: Container Queries animadas

Animações responsivas ao container, não à viewport. Use unidades cqi/cqw e @container queries. Chrome 105+

Card Responsivo (arraste para redimensionar)

Container Title

Texto que se adapta ao tamanho do container, não da viewport.

.card {
  container-type: inline-size;
  container-name: card;
}
.title { font-size: 14cqi; }
.text  { font-size: 3.5cqi; }

@container card (min-width: 300px) {
  .title { font-size: 10cqi; }
  .text  { font-size: 2.8cqi; }
}
@container card (min-width: 450px) {
  .title { font-size: 8cqi; }
  .text  { font-size: 2.2cqi; }
}

Dot Animado Responsivo

Animação muda com tamanho
.card {
  container-type: inline-size;
}
.dot {
  width: 8cqi;
  height: 8cqi;
  animation: spin 2s linear infinite;
}
@container (min-width: 250px) {
  .dot {
    animation-duration: 1s;
    /* gira mais rápido em
       containers maiores */
  }
}

Hover + Container

⚡

Hover ou resize

.card {
  container-type: inline-size;
}
.icon {
  font-size: 10cqi;
  transition: transform 400ms;
}
.card:hover .icon {
  transform: scale(1.2) rotate(15deg);
}

Style Queries (CSS Conditional Level 5)

Hover muda a custom property --dark, e a style query reage

@property --dark {
  syntax: '<number>';
  initial-value: 0;
  inherits: true;
}
.card {
  container-type: inline-size;
  --dark: 0;
  transition: --dark 300ms ease;
}
.card:hover { --dark: 1; }

@container style(--dark: 1) {
  /* reage à custom property */
  .text { font-weight: 700; }
}

Unidades cqi/cqw/cqh

Container query length units, relativas ao container pai.

cqi  /* 1% do inline-size do container */
cqb  /* 1% do block-size do container */
cqw  /* 1% do width (deprecated alias) */
cqh  /* 1% do height (deprecated alias) */

.container {
  container-type: inline-size;
  /* ou 'size' para ambos */
}
.element {
  font-size: 5cqi;
  padding: 2cqi;
  width: 80cqi;
  height: 30cqb;
}
